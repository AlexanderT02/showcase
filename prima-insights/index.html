PRIMA Insights

Internal sales intelligence platform used by a hospitality supplier to activate dormant customer demand and structure follow-ups across the sales team.

Built a sales activation system that turns passive shop behavior into actionable leads. Eliminated manual follow-ups and enabled agents to systematically recover missed revenue opportunities.

Impact

  • Used weekly by the sales team
  • Replaced manual follow-up workflows
  • Turned shop analytics into actionable leads
  • Centralized customer interaction visibility

Core Flow

Weekly Lead Activation

Automated weekly emails highlight customer interest without purchase. Each email links directly to the dashboard, where agents can filter assigned companies and review detailed insights before initiating follow-up.

Problem

Shop behavior data existed, but wasn’t operationally accessible for sales. Follow-ups relied on manual communication and personal knowledge. Goal: convert passive events into assigned, low-noise actions that integrate into a weekly workflow.

Approach

PRIMA Insights transforms customer behavioral data into actionable sales workflows and generates automated sales activation pipeline whenever relevant activity occurs. Each notification links directly to a filtered dashboard showing the responsible accounts and their detailed interactions. This turns passive shop activity into a structured follow-up workflow.

System Context

Production microservice architecture composed of a core processing service and a dedicated web application backend. The core service performs data transformation, behavioral analysis and automated email delivery, while the web backend handles authentication, role management and dashboard interaction.
